# 배열 전개 연산자와 객체 전개 연산자
## 배열 전개 연산자

### 1. 얕은 복사
배열 1의 값을 배열 2에 넣을 때:
```javascript
const array1 = ["사과", "복숭아", "딸기"]; 
const array2 = array1;
이렇게 하면 복사가 이루어지지만, 다음과 같이 
array2.push("망고");
console.log(array1); // ["사과", "복숭아", "딸기", "망고"]
console.log(array2); // ["사과", "복숭아", "딸기", "망고"]
두 배열이 동일한 데이터를 가지게 됩니다. 이는 얕은 복사(참조 복사)이기 때문이다.
2. 깊은 복사
독립적으로 사용하고 싶다면 깊은 복사를 사용해야 한다. 전개 연산자를 이용하면 된다:
const array1 = ["딸기", "복숭아", "사과"];
const array2 = [...array1];

const array2 = ["망고", ...array1, "체리"];
const array3 = [...array1, ...array2];

객체 전개 연산자

1. 기본 사용법
전개 연산자를 사용하여 객체 복사:
const 구름 = {
  이름: "구름",
  나이: 1,
  종족: "강아지"
};

const 별 = { ...구름 };
별.이름 = "별";
별.나이 = 5;

2. 자료 추가 및 대입
다음과 같이 기존 객체에 새로운 데이터를 추가할 수 있다:
const 별 = {
  ...구름,
  이름: "별",
  나이: 5,
  예방접종: true
};
